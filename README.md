# Shu

# 【職務経歴書・スキルシート】

はじめまして。Webコーダーのしゅうと申します。 20年以上の接客・カスタマーサポートの経験を活かし、クライアント様や制作チームの皆様が安心してプロジェクトを進められるよう、コミュニケーションを大切に活動しています。

## ◆ 稼働時間・連絡手段

〇 稼働時間 ・週30時間〜50時間（平日・土日関係なく対応可能です）

〇 連絡手段 ・Slack / Chatwork / Zoom / LINE など、ご希望のツールで柔軟に対応します。

- 原則、メッセージには即レスを心がけており、遅くとも3時間以内には返信します。

## ◆ 私ができること・強み

〇 クライアント様の工数を削減する「先回り力」

長年のCS経験から、指示書の不明点を早めに確認したり、実装上の懸念点を事前に提案したりすることで、手戻りの少ないスムーズな進行をサポートします。

〇 デザインの意図を汲み取ったコーディング

単に見た目を合わせるだけでなく、ボタンのホバーアクションやスクロール時の動きなど、ユーザーが心地よく感じる実装を意識しています。

〇 メンテナンス性を重視した設計

「BEM」などに基づいた命名規則を採用し、後からの修正や機能追加がしやすい、他の方が読んでも理解しやすいコードを書くようにしています。

## ◆ 職務要約

〇 カスタマーサポート・接客業務（1998年〜2024年）
20年以上にわたり、大手通信会社のサポート業務や対面接客に従事してきました。
複雑な仕様を分かりやすく説明する言語化能力や、トラブルを未然に防ぐ先回りした提案力を磨きました。この経験が、現在の制作業務におけるスムーズなヒアリングと確実な納期管理の土台になっています。

〇 Webコーディング業務（2024年〜現在）
実案件の制作に加え、WordPressを用いたサイト構築やLPの実装を中心に行っています。
単にデザインを形にするだけでなく、運用担当者が更新しやすい設計（カスタムフィールドの活用や、部品のコンポーネント化等）を意識した実装をしています。
